      • Vanila JS 를 이용한 Momentum 입니다. 만든이 : JhsK (qwe6293@nate.com)
      • 해야할 일정, 계획 등을 추가하고 삭제 할 수 있는 일정관리 페이지입니다.
      • 버그 및 오류, 개선할 점 등이 있으시다면 언제든지 해당 이메일(qwe62932nate.com)으로 요청 부탁드립니다.
      -사용 방법-
      1. "이름을 입력해주세요" 라고 보이는 입력 창을 클릭 후 사용자의 이름을 입력해주세요.
      2. "할 일을 입력해주세요" 라고 보이는 입력 창을 클릭 후 앞으로 해야 할 계획 등을 입력 후 엔터(Enter)를 눌러주세요 자동으로 Schedule 창으로 입력하신 계획이 추가 됩니다.
      3. 언제든 계획을 삭제하고 싶다면 추가된 계획 옆에 보이는 ❌ 버튼을 클릭하여 삭제해주세요.
      4. Schedule 창에 있는 계획을 완료하셨다면 계획 옆에 보이는 ✅ 버튼을 클릭하여 Finished 창으로 이동 가능합니다.
      5. Finished 창에 있는 완료한 계획을 다시 계획 예정인 Schedule 창으로 이동하고 싶다면 ⏪ 버튼을 클릭하여 이동해주세요.
